# Sistema de Apadrinhamento Afetivo

## 📌 Descrição

Este projeto consiste no desenvolvimento de um banco de dados para um sistema de apadrinhamento afetivo de crianças em instituições de acolhimento.

A plataforma tem como objetivo conectar usuários interessados em ajudar com ONGs responsáveis por crianças, facilitando o processo de apadrinhamento e divulgação.

---

## 🎯 Objetivo

* Facilitar o apadrinhamento afetivo
* Divulgar crianças disponíveis em abrigos
* Centralizar informações em um sistema estruturado
